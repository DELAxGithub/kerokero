# IELTS Annotation System (Reference for Future Evaluator Enrichment)

> NOT for Day 1. Save for Days 5-10 when enriching evaluator feedback with
> annotation categories and band-level markers.

```
※アノテーションexplanationにIELTS7.5 相当の類似表現も追記する

Convert the following IELTS essay into a structured JSON format for language learning.

INSTRUCTIONS:
1. Divide the essay into segments based on paragraph breaks
2. For each segment, identify and annotate 8-12 key expressions that are transferable to other IELTS topics
3. Focus on expressions that demonstrate Band 7+ writing skills
4. Create practical reading guides for learners

ANNOTATION CATEGORIES:
- backgrounding: Setting context/background
- introduction: Previewing essay structure
- opinion: Expressing personal views
- reason: Giving causes/explanations
- example: Providing illustrations
- contrast: Showing opposition/alternative views
- conclusion: Summarizing/concluding
- emphasis: Highlighting key points
- result: Showing consequences
- collocation: High-level word combinations
- paraphrase-potential: Alternative expressions for common ideas

ANNOTATION PRIORITIES:
1. Expressions that can be reused across different IELTS topics (high transferability)
2. Sophisticated alternatives to common expressions (paraphrase-potential)
3. Expressions that signal logical relationships (contrast, result, reason)
4. Academic collocations that demonstrate Band 8+ vocabulary
5. Phrases that show nuanced thinking (emphasis, sophisticated opinions)
```
